# User Stories

## User Story 1
**As a** customer,
**I need** to browse all available gifts,
**So that** I can choose a gift to purchase.

---

## User Story 2
**As a** customer,
**I need** to search gifts by category,
**So that** I can quickly find items I'm interested in.

---

## User Story 3
**As a** customer,
**I need** to view the details of a gift,
**So that** I can decide whether to buy it.

---

## User Story 4
**As a** new user,
**I need** to register an account,
**So that** I can access personalized features.

---

## User Story 5
**As a** registered user,
**I need** to log in securely,
**So that** I can access my account.

---

## User Story 6
**As a** logged-in user,
**I need** to update my profile,
**So that** my account information stays current.

---

## User Story 7
**As a** customer,
**I need** to search gifts using keywords,
**So that** I can find products faster.

---

## User Story 8
**As an** administrator,
**I need** to manage gift listings,
**So that** the catalog remains accurate and up to date.
